# 🚀 CareerPilot AI

An intelligent AI assistant built using Streamlit + Ollama that delivers structured, context-aware answers instead of random chatbot responses.

---

## 🧠 What This Project Does

CareerPilot AI improves normal AI chat by:

- Understanding user intent (explain, compare, plan)
- Structuring responses (Insight → Why → Example → Judgment)
- Maintaining context across conversation
- Generating dynamic follow-up questions
- Avoiding repetitive / generic outputs

👉 Result: More focused, useful, and human-like answers

---

## ⚙️ Tech Stack

- Python
- Streamlit (UI)
- Ollama (LLM - Mistral)

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Start Ollama:
ollama serve

3. Pull model:
ollama pull mistral

4. Run app:
streamlit run app.py
